Faunatowers

Faunatowers

Following the minor "Emerging technologies playground" (ETP) we worked in the period of February 2021 to July 2021, in conjunction with the municipality of den bosch to create a monitoring system which uses machine learning to determine which specie of bird is flying into a fauna tower. A faunatower is a large tower which can harbor animals such as birds that are dissapearing due to houses being better isolated and leaving little to no cracks for these birds to shelter. The final product made use of a machine learning algorithm called Birdnet and we implemented it in our Angular application and wrote a wrapper for it using C++/C and Python.
